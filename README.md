# Mesto
"Mesto" - это проект, представляющий собой упрощенный аналог Instagram. Этот веб-сайт предлагает пользователям возможность делиться фотографиями посещенных мест, ставить лайки и удалять карточки. Проект разработан для демонстрации умений работы с Webpack и Babel, а также для применения современных подходов к разделению кода на компоненты.

# Функционал
Сайт "Mesto" представляет собой веб-платформу, которая включает в себя следующий функционал:  
Обновление профиля пользователя: позволяет изменить имя пользователя и его описание.  
Добавление фотографий: пользователи могут загружать изображения мест, которые они посетили.  
Лайки: возможность выражать нравится ли вам фотография, поставив ей "лайк".  
Удаление карточек: пользователи могут удалять карточки с фотографиями, которые они ранее добавили.  
Код организован по принципу компонентности: структура кода организована таким образом, что каждый компонент отвечает за определенную функциональность и имеет свои собственные зависимости, что обеспечивает удобство поддержки и масштабирования проекта.

# Возможные улучшения
Добавление комментариев: ввод системы комментариев, позволит пользователям общаться и обмениваться мнениями о различных местах.  
Мультиязычность: в настоящее время сайт доступен только на русском языке. Добавление английской версии сайта сделает его доступным для большего числа пользователей.
